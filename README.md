# ny-soundroom

Standalone GitHub Pages page for a YouTube-based Sound Room.

## Deploy

1. Create a new public repository named `ny-soundroom` under `tomoponz`.
2. Upload these files to the repository root.
3. In GitHub: Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / root.
4. The page should be available at:

https://tomoponz.github.io/ny-soundroom/

This page uses the official YouTube embed only. It does not copy audio, lyrics, official thumbnails, or MV assets.
