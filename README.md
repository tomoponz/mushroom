# mushroom

A standalone experimental page for `https://tomoponz.github.io/mushroom/`.

## Concept

子供向けのキノコ動画サムネイルから始まり、再生ボタンを押すと、文字化け・菌類・ネットミーム・ホラー寄りの異界ページへ切り替わる実験ページ。

## Videos

- Initial thumbnail: `iH0ze-eg54Y`
- Main embed: `NYnpsWrVGKg`

The page uses YouTube official thumbnail/embed URLs. It does not copy audio, lyrics, or official video files.

## Optional assets for future upgrades

The current version works without local image assets. If you want to upgrade the page later, add assets such as:

```txt
assets/img/mushroom-cute-bg.webp
assets/img/mushroom-nightmare-bg.webp
assets/img/mushroom-face.webp
assets/img/glitch-noise.webp
assets/img/eye-cutout.webp
assets/img/mouth-cutout.webp
assets/img/fungal-wall.webp
```
